# Positive Test examples are below write test process steps for each below test case

Create blank project
	Navigate To Project Page     Blank_Project_Url
	Select Create Blank Project
	Input Project Name           Project_Name
	Input Project Description    Project_Description
	Select Visibility            Visibility_Private
	Click Create Project
	Verify Page Should Contain   Project 'First project' was successfully created.
	Verify Project Visible In    user project list

Create template project
	Navigate To Project Page     Template_Project_Url
	Select Create From Template
	Choose Template              Default Template
	Input Project Name           Template_Name
	Click Create Project
	Verify Page Should Contain   Project 'Template project' was successfully created.
	Verify Project Visible In    user project list

Update name of template project
	Navigate To Existing Project Template_Name
	Open Project Settings
	Update Project Name          Updated_Project_Name
	Save Changes
	Verify Page Should Contain   Project 'First project Updated' was successfully updated.
	Verify Project Visible As    Updated_Project_Name

Delete template project
	Navigate To Existing Project Updated_Project_Name
	Open Project Settings
	Click Delete Project
	Confirm Delete
	Verify Page Should Contain   Project was successfully deleted.
	Verify Project Not In        user project list
